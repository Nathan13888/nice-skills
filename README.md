# Nice Skills

Opinionated (read "objectively correct") skills for coding agents like Claude Code, Codex, Antigravity and other agents.

## Available Skills

- `/init-repo`: Initialize a new project repository for development and (optionally) create a MVP for your project.
- `/mvp`: Product design and feature planning for new projects -- deep requirements gathering, feature ideation, scope management, technical architecture, and living documentation.
- `/sus`: Find sloppy and highly problematic code, affecting maintainability and core functionalities of the application.
- `/check`: Confirm something specific you expected is correct and done to your satisfaction.
- `/explore`: Strategic discovery of a project's capabilities -- catalogs existing features and identifies improvements and expansion opportunities.
- `/reflect`: Ask the agent to think about what it has done and if it's done a good job itself.
- `/wtf`: Quick situational awareness for the current git branch -- what is this branch about, or what's been happening on trunk.
- `/combine-docs`: Consolidate non-code based documentation in the repository so all existing
